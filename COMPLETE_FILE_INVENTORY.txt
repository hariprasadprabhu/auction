🚀 QUICK START - FOLLOW THESE 8 STEPS
================================================================================
Step 1: Build Locally (2-3 minutes)
Command: mvn clean package -DskipTests
Result: Creates target/auction-0.0.1-SNAPSHOT.jar
Step 2: Get Neon Connection String (2 minutes)
Go to: https://console.neon.tech
Copy: PostgreSQL connection string
Example: postgresql://user:password@ep-xxxxx.neon.tech/auctiondeck
Step 3: Push to GitHub (1 minute)
Commands:
  git add .
  git commit -m "chore: prepare for Railway deployment"
  git push origin main
Step 4: Deploy Backend on Railway (10-15 minutes)
1. Go to https://railway.app
2. Create new project
3. Deploy from GitHub repo
4. Select your auction repository
5. Wait for build and deployment
Step 5: Set Environment Variables (5 minutes)
Set in Railway Dashboard:
  DATABASE_URL=postgresql://user:password@...
  CORS_ORIGINS=http://localhost:4200
  (and others - see guides for complete list)
Step 6: Deploy Frontend (5-10 minutes)
1. Create new service in Railway
2. Choose Static Site
3. Build: npm install && npm run build
4. Publish: dist/auction
Step 7: Update CORS (2 minutes)
Update CORS_ORIGINS to your frontend URL:
  https://auction-web.railway.app
Step 8: Test (5-10 minutes)
Test backend: curl https://auction-api.railway.app/api/health
Test frontend: Open https://auction-web.railway.app
Check Network tab for API calls
