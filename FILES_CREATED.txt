ENVIRONMENT VARIABLES TO SET IN RAILWAY
========================================
Copy and paste all of these into Railway Dashboard → Backend Service → Variables:
DATABASE_URL=postgresql://user:password@ep-xxxxx.neon.tech/auctiondeck
CORS_ORIGINS=http://localhost:4200
PORT=8080
LOG_LEVEL=INFO
COM_BID_AUCTION_LOG_LEVEL=DEBUG
JWT_SECRET=YourSuperSecretKeyHereMustBeAtLeast256BitsLongForHmacSha256!!
JWT_EXPIRATION=86400000
JPA_HIBERNATE_DDL_AUTO=validate
JPA_SHOW_SQL=false
⚠️ IMPORTANT: Replace the DATABASE_URL with your actual Neon connection string!
QUICK START
===========
1. Build locally:
   mvn clean package -DskipTests
2. Get your Neon connection string from: https://console.neon.tech
3. Push to GitHub:
   git add .
   git commit -m "chore: prepare for Railway deployment"
   git push origin main
4. Go to https://railway.app and create a project
5. Deploy from GitHub repository
6. Set the environment variables above
7. Test your app! 🎉
